# Task Manager API

This is a CRUD (Create, Read, Update, Delete) project built with Node.js and Express. It serves as a learning project to explore and understand the fundamentals of building a backend application using Node.js, Express, and a simple API structure.

## Features
- Task creation, reading, updating, and deletion functionality
- Simple, intuitive REST API

## Technologies Used
- Node.js
- Express.js
- JavaScript (ES6+)
